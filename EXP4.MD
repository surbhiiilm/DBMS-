# EXPERIMENT 4

## 1.Display the list of employees who have joined the company before 30th june 80 or after 31st dec 81.
```sql
 SELECT * FROM EMPLOYEE
    -> WHERE HIREDATE < "1980-06-30"
    -> OR HIREDATE > "1981-12-31";
```
## 2. Display the names of employees whose names have second alphabet A in their names.
```sql
 SELECT * FROM EMPLOYEE
    -> WHERE ENAME LIKE "_A%";
```

## 3. Display the number of employees whose names is exactly five characters in length.
```sql
 SELECT * FROM EMPLOYEE
    -> WHERE ENAME LIKE "_____";
```
## 4. Display the names of employees whose names have second last alphabet E in their names.
```sql
SELECT * FROM EMPLOYEE
    -> WHERE ENAME LIKE "%E_";
```
## 5. Display the names of employees who are not working as salesman or clerk or analyst.
```sql
SELECT ENAME FROM EMPLOYEE
    -> WHERE JOB NOT IN('SALESMAN','CLERK','ANALYST');
```
## 6. Display the names of the employee along with their annual salary (sal*12). The name of the employee earning highest salary should appear first.
```sql
SELECT ENAME,(SAL*12) AS ANNUAL_SALARY
    -> FROM EMPLOYEE
    -> ORDER BY ANNUAL_SALARY DESC;
```
## 7. Display name,sal,hra,pf,da,totalsal for each employee. The output should be in the order of total sal,hra 15% of sal,da 10% of sal, pf 5% of sal. Total salary will be (sal*hra *da) -pf.
```sql
SELECT ENAME,
    -> ((SAL) + (SAL * 0.15) + (SAL * 0.10)) - (SAL * 0.05) AS TOTAL_SALARY,
    -> (SAL * 0.15) AS HRA,
    -> (SAL * 0.10) AS DA,
    -> (SAL * 0.05) AS PF
    -> FROM EMPLOYEE;
```
## 8. Update the salary of each employee by 10% increment who are not eligible for commission.
```sql
 UPDATE EMPLOYEE
    -> SET SAL = SAL * 1.10
    -> WHERE COMM IS NULL;
```
## 9. Display those employees whose slary is more than 3000 after giving 20% increment.
```sql
SELECT * FROM EMPLOYEE
    -> WHERE SAL * 1.20 > 3000;
```
## 10. Display those employees whose salary contains at least 3 digits.
```sql
 SELECT * FROM EMPLOYEE
    -> WHERE SAL BETWEEN 100 AND 999;
```
